# ADR-002 Azure Authentication

## Estado

Aceptado provisionalmente.

## Decisión

Usar PAT para desarrollo y MVP interno.

## Condiciones

- Permisos mínimos.
- Almacenamiento backend.
- Rotación.
- Nunca en frontend.
- Nunca en logs.

## Evolución

Migrar a OAuth de Microsoft Entra ID o Service Principal.
